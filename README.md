# 🎓 OUSL Student Hub - Professional Dashboard

A high-performance, responsive academic management dashboard tailored for the Open University of Sri Lanka (OUSL). This platform provides specialized tools designed to streamline the academic experience for Science undergraduates.

## 🚀 Core Features
- **Advanced GPA Calculator:** Custom-built logic supporting 13+ Level 5 modules (Physics & Chemistry), providing accurate, real-time academic tracking.
- **Academic Interface:** Clean, intuitive UI for managing course materials, schedules, and university notices.
- **Responsive Architecture:** Fully optimized for seamless performance across mobile and desktop environments.
- **Modern Tech Stack:** Engineered with Next.js and TypeScript for type-safe, high-speed execution.

## 🛠️ Technical Specifications
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **Deployment:** Vercel

## ⚙️ Local Setup
To run this project locally:
```bash
git clone [https://github.com/SANKA-OUSL/ousl-student-hub-pro.git](https://github.com/SANKA-OUSL/ousl-student-hub-pro.git)
cd ousl-student-hub-pro
npm install
npm run dev
